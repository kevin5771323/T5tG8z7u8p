# 项目简介

## 核心功能
- 基于YOLOv8的钢材表面缺陷检测系统
- 支持2700张钢材表面缺陷数据集，包含6类缺陷：开裂、内含杂质、斑块斑点、点蚀表面、轧制氧化皮、划痕
- 支持多种输入方式检测：单张图片、图片文件夹、视频文件、本地摄像头
- 提供自定义设置：置信度阈值、IOU阈值、显示标签和原图选择、检测模型选择
- 实时显示检测信息：目标位置、目标总数、置信度
- 支持批量检测及结果查看
- 支持检测结果保存

## 适用场景
- 钢材生产质量检测
- 工业自动化生产线缺陷识别
- 材料表面质量分析
- 制造业质量控制与监控

## 技术特点
- 软件环境：Pycharm+Anaconda或VSCode+Anaconda
- 技术栈：Python 3.9、OpenCV-Python、PyQt5、Ultralytics、PyTorch 1.9
- 提供完整源码、标注数据集、训练模型、运行教程
- 包含UI界面源文件及图标
- 已训练好模型，配置成功环境可直接使用
